# Laniakea Phase 1 Rules

## Scope
- No auth, no GitHub integrations, no functional frontend
- Engine: XP calculation, streaks, achievements, leaderboard
- API: REST endpoints, MongoDB models, service layer
- Web: React shell with routing placeholder only

## Constraints
- Do NOT modify legacy Go code (src/, scripts/, Makefile, README.md)
- Do NOT run git add/commit/push
- All TypeScript must pass strict mode with zero `any`
- All Rust must pass clippy pedantic with zero warnings
- Tests required for all business logic
